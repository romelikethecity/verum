## Google Search Console Setup

1. Go to https://search.google.com/search-console
2. Add property: `veruminc.com`
3. Verify via DNS TXT record in Cloudflare
4. Submit sitemap: `https://veruminc.com/sitemap.xml`
5. Monitor indexing and coverage

## Common Issues to Avoid

1. **Wrong canonical domain** - Always use `veruminc.com`, never GitHub Pages URL
2. **Missing OG tags** - Pages won't preview properly on LinkedIn/Twitter
3. **Duplicate content** - Each page needs unique title/description
4. **Broken internal links** - Check after regenerating pages
5. **Missing tracking** - Every page needs GA4 + Clarity code
